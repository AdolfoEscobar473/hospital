Logo corporativo - SGI Hospital
================================

El archivo logo-hospital.psd está en esta carpeta. Para que el logo se vea en la aplicación (login y barra lateral):

1. Abre logo-hospital.psd en Photoshop (o GIMP, etc.).
2. Exporta o guarda como imagen para web:
   - Formato: PNG o JPG
   - Nombre: logo-hospital.png (o logo-hospital.jpg)
   - Guarda en esta misma carpeta (public).

La aplicación busca primero logo-hospital.png y luego logo-hospital.jpg.
Si no existe ninguno, se muestra un icono por defecto.
